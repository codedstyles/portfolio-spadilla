Life360 LP

Client: Life360 | Category: Design & Code

Overview

Life360 is a location-based service designed primarily to allow friends and family members share their locations with each other. I worked with the Marketing team in developing a Landing Page that will help increase their website's conversion rate and lower the cost of acquiring leads.

Layout

When defining the content, I followed Life360's Style Guide, which included a structured color palette, typeface and imagery guidelines. For the page structure, I worked with a responsive format, using Bootstrap's grid system.

Build

After final approval by the client, I built the Landing Page in Bootstrap, using the CSS Flexbox layout module for a a one-dimensional layout model, which offered accurate alignment between elements, enabiling the design translate pixel perfect.
