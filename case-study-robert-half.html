Robert half

Client: Robert Half | Category: UI/UX Design

Overview

The project entailed the re-design of Robert Half's global menu by creating a navigation system that would guide the user in the most efficient manner to the main sections of the website.

Objectives

After in-depth research, we became aware that there were usability issues with the current menu system. The user was also having difficulty in finding the content they were searching when on the site. These navigation problems became our two main objectives.

User Flow

In the User Flow we can observe how we worked in separating the two main users - Job Seekers and Employers, making content available depending on the type of user. Also in this stage, we were able to categorize and reduce the content displayed in the new navigation.

Wireframes

Wireframes were created, to explore different menu options as possible solutions.

Results & Implementation

Our rounds of iterations clearly showed that 90% of our users used the main navigation to search for a specific section of the website. They did not use the site search, which was only used when the navigation did not display all content. Testing also showed that most of the time users did not browse, as users are defined as being task orientated.

For this project we worked with the Sprint methodology, which lead us to perform various types of testing from the very first weeks, which helped us advance with our early assumptions and agree on a best solution for the website's navigation.
